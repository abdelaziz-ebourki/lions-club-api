# lions-club-api

REST API for Lions Club FSBM, built with Spring Boot 3 + Java 21.

## Prerequisites

- Java 21
- Docker (for PostgreSQL)

## Getting Started

```bash
# Start the database
docker compose up -d

# Run the application
./mvnw spring-boot:run
```

## API Documentation

Once running, visit http://localhost:8080/swagger-ui.html
